This website is a SAAS (software as a service) application landing page for a customizable meal plan product that the users can purchase. The app gives the user the ability to create unique meal plans that cater to their specific life style and dietary needs.

They landing page is simple and clean and provides a description of what the product is and provides a list of the key features that the app provides.

The pricing page provides the user with different pricing options with different levels of depth to them depending on how much the user is willing to pay for the app.
